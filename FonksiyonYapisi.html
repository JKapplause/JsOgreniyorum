Bir JavaScript işlevi function, ardından bir ad ve ardından parantez () ile birlikte tanımlanır .
İşlev adları harf, rakam, alt çizgi ve dolar işareti içerebilir (değişkenlerle aynı kurallar).
Parantezler virgülle ayrılmış parametre adları içerebilir:
( parametre1, parametre2, ... )

İşlev tarafından yürütülecek kod, küme parantezlerinin içine yerleştirilir: {}

function name(parameter1, parameter2, parameter3) {
// code to be executed
}

İşlev parametreleri , işlev tanımındaki parantez () içinde listelenir.
Fonksiyon argümanları olan değerler o çağrıldığında işlevi tarafından verildi.
İşlevin içinde, bağımsız değişkenler (parametreler) yerel değişkenler gibi davranır.

**Bir İşlev, diğer programlama dillerinde bir Prosedür veya Alt Program ile hemen hemen aynıdır.
